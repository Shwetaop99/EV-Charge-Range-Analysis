EV Charge Range Analysis

This project analyzes Electric Vehicle data using Tableau dashboards.

Project Contents

- EV_Cars_Analysis_Dashboard.twbx – Main Tableau dashboard
- EVIndia.csv – EV dataset for India
- ElectricCarData_Clean.csv – EV specifications dataset
- electric_vehicle_charging_station_list.csv – Charging stations data
- Cheapestelectriccars-EVDatabase.csv – EV price dataset

Tools Used

- Tableau
- CSV Datasets
- GitHub

The dashboard provides insights into EV performance, charging infrastructure, pricing, and efficiency trends.
